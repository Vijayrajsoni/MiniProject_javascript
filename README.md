# 🚀 JavaScript Mini Projects Collection

A collection of beginner-friendly JavaScript projects built using **HTML**, **CSS**, and **JavaScript**. These projects are designed to improve fundamental web development skills through hands-on practice.

## 📂 Projects Included

- 🎂 Age Checker
- 🧮 Calculator
- 💱 Change Calculator
- ⏳ Countdown Timer
- 🔢 Counter
- 🎯 Guess Number Game
- 🎵 Know Your Sound
- 🔊 Sound Player

Each project is independent and demonstrates different JavaScript concepts such as:

- DOM Manipulation
- Event Handling
- Functions
- Variables
- Loops & Conditions
- Timers (`setInterval`, `setTimeout`)
- User Input Validation
- Audio Handling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

1. Clone the repository

```bash

```

2. Open the project folder.

3. Navigate to any project directory.

4. Open the `index.html` file in your browser.

No installation or dependencies are required.
